আপনার প্রজেক্ট ফ্রিতে হোস্ট করুন

pythonanywhere.com আপনাকে আপনার পাইথন প্রোজেক্ট ফ্রিতে হোস্ট করতে দেয় | আপনাকে মোট 512.0 MB ডাটা রাখতে দিবে আর আপনি যদি পেইড কাস্টমার হয়ে থাকেন তাহলে তো কোন কথাই নেই |

PythonAnywhere এ কোন প্রজেক্ট Deploy করা PC এর লোকালহস্টে প্রোজেক্ট রান করার মতই | আপনি এখানেও ভারচুয়াল ইনভাইরনমেন্ট বানাতে পারবেন,চাইলে কোড এডিট করতে পারবেন , গিটে কমিট করতে পারবেন | সবচেয়ে মজার বেপার হল,Dnajgo প্রজেক্ট লোকালহস্টে রান করার জন্য যেই কমান্ডগুলো দিতেন এখানের কমান্ডগুলোও প্রায় সেইম |
তো ঝটপট এখান থেকে একাউন্ট খুলে ফেলুন ...

শুরু করুনঃ

console এ গিয়ে Bash console ওপেন করুন , এটা আপনার টার্মিনাল এর মতই

প্রোজেক্ট আপলোড করুনঃ

নিচের কমান্ড দিন | উল্লেখ্য git clone এর পরে যেই Url টা আছে সেটা আপনি যেই প্রোজেক্ট টা pythonanywhere এ হোস্ট করতে চাচ্ছেন তার Url |

$ git clone https://github.com/myusername/myproject.git

ভারচুয়াল ইনভাইরনমেন্ট তৈরি করুন এবং Django ইন্সটল করুনঃ

$ mkvirtualenv --python=/usr/bin/python3.6 myvenv
(myvenv)$ pip install django

বুঝতেই পারছেন আমি আমার প্রোজেক্ট এর জন্য Python3.6 ইন্সটল করেছি আপনি যেটা নিয়ে কাজ করেছেন সেটা দিবেন , আর আমি আমার ভারচুয়াল ইনভাইরনমেন্টএর নাম দিয়েছি myvenv, আপনার পছন্দ মত আপনি যেকোনো নাম দিন যদু,মদু ইত্যাদি | তারপর Django ইন্সটল করার জন্য পরবর্তী কমান্ডটি দিয়েছি |
Django ইন্সটল করতে একটু সময় লাগবে কারণ Django এর অনেক ছোট ছোট ফাইল থাকে তাই একটু অপেক্ষা করুন |
ভারচুয়াল ইনভাইরনমেন্ট তৈরি করতে সমস্যা হলে অথবা mkvirtualenv: command not found এই এরর থ্রো করলে নিচের কমান্ডগুলো দিন , তারপর Django ইন্সটল করুন |

echo '' >> ~/.bashrc && echo 'source virtualenvwrapper.sh' >> ~/.bashrc
source virtualenvwrapper.sh

এরপর ফাইল ট্যাবে ক্লিক মারে একটু দেখে আসুন কি কি পরিবর্তন হয়েছে , কি কিছু দেখতে পেলেন ?

Web app সেটাপ করুনঃ


Web ট্যাবে ক্লিক করুন তারপর Add a new web app এ ক্লিক করুন তারপর Next এ ক্লিক করুন অর্থাৎ Web>>Add a new web app>>Next অনুসরণ করুন , নিচের মত উইন্ডো আসবে |

আমরা ম্যানুয়ালি আমাদের Web app সেটাপ করবো তাই Manual Congiration(including virtualenvs) এ ক্লিক করুন তারপর Python 3.6 এ ক্লিক করুন |

ওভারঅল Web app কনফিগারেশন এর জন্য নিচের পদ্ধতি অনুসরণ করুন Web>>Add a new web app>>Manual Congiration>>Python 3.6 |

virtual environment সেটাপ করুনঃ

দেখতেই পাচ্ছেন আমি আমার virtual environment এর নাম পূর্বে myvenv দিয়েছিলাম এখানেও তাই দিয়েছি | আপনি আপনার virtual environment এর যেই নাম দিয়েছিলেন সেই নাম দিয়ে টিক এ ক্লিক করুন |

Static ফাইলের জন্য সেটাপ করুনঃ

এই অংশে আমি URL হিসাবে /static/ দিয়েছি এবং ডিরেক্টরির পাথ হিসাবে
/home/sharifthesimple/Dream-palace/static দিয়েছি
এখানে,
home -> আমার ডিরেক্টরি |
sharifthesimple -> আমার কর্তৃক প্রদত্ত নাম |
Dream-palace -> যেই প্রজেক্টটি হোস্ট করতে চাচ্ছি সেটা |
static -> আমার স্ট্যাটিক ফাইলের ডিরেক্টরি নাম |

WSGI ফাইল কনফিগার করুনঃ

WSGI configuation file এ ক্লিক করুন ইহা আপনাকে একটি কোড এডিটর এ নিয়ে যাবে | অনেকগুলো সেকশন দেখতে পাবেন, শুধুমাত্র Django সেকশন বাদে বাকি সব ডিলিট করুন |
আমি আমার প্রজেক্ট এর জন্য WSGI file নিচের মত কনফিগার করেছি , আপনিও একই ভাবে কনফিগার করুন | শুধুমাত্র Path টার ক্ষেত্রে একটু পরিবর্তন করুন আমি Path হিসাবে path = '/home/sharifthesimple/Dream-palace' দিয়েছি কেন দিয়েছি সেটা এর আগেই একবার ব্যাখ্যা করেছি এখানে তাই আর ব্যাখ্যা করার প্রয়োজন বোধ করছি না |

তারপর আপনার URL এ ক্লিক করুন | সব ঠিকঠাক থাকলে আশা করি আপনার প্রোজেক্ট অনলাইনে হোস্ট হয়েছে আলহামদুলিল্লাহ |

কিছু সমস্যাঃ

$ python manage.py collectstatic
