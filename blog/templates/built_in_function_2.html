বিল্ট ইন ফাংশন পার্ট ২

এই পর্বে আমরা Python এর কিছু বিল্ট ইন ফাংশন এর এর সাথে পরিচিত হবো | Python যতই দিন যাচ্ছে ততই জনপ্রিয় হচ্ছে এর অনেক গুলো কারন এর মাঝে যেই দুইটা সবার আগে আসে তা হল এর সহজবোদ্ধতা এবং এর সুবিশাল লাইব্রেরী |
এবার একটু সামনের দিকে এগুনো যাক ..........

map():

মূলত এই মেথডের সিনটেক্সট টা এমন map(function,iterable1, iterable2,....) |
মূলত এই মেথডে একটা ফাংশনের ভিতর দিয়ে এক বা একাধিক ইটারেবল পাস করা হয় এবং এই ইটারেবল এর উপর ফাংশনটি apply করা হয় | কিছু example দেখলে বিষয় গুলো আরো পরিষ্কার হবে ,

def squre(n):
>>>return n*n

s=[1,2,3,4,5]
print(list(map(squre,s)))
Output : [1, 4, 9, 16, 25, 36]

num1 = [4, 5, 6]
num2 = [5, 6, 7]
result = map(lambda n1, n2: n1+n2, num1, num2)
print(list(result))
Output : [9, 11, 13]

filter():

নাম থেকেই বোঝা যাচ্ছে এর কাজ ফিল্টারিং করা | এর সিনটেক্স এবং কাজের ধরন অনেকটাই map() মেথডের মতই | কিছু কন্ডিশন এর উপর ভিত্তি করে ইটারেবল গুলোর উপর ফিল্টারিং এপ্লাই করা |

num=[1,-2,3,4,5,-10]
def positive(n):
>>>if n>=0:
>>>return n

print(list(filter(positive,num)))
Output: [1, 3, 4, 5]

নিচের example টি একটি নাম্বার জোড় কিনা তা চেক করবে |

def even(n):
>>>if n%2==0:
>>>return n

print(list(filter(even,num)))
Output: [-2,4,10]

reversed():

এই মেথডের মাধ্যমে কোন একটা লিস্ট এর আইটেম গুলোকে রিভারস করা যায় |

lst=[1,2,3,10,5,6]
print(list(reversed(lst)))

Output: [6, 5, 10, 3, 2, 1]

একই ভাবে একটা স্ট্রিং এর আইটেম গুলোকে রিভারস করতে পারি |

s = 'Python'
print(list(reversed(s)))

Output: ['n', 'o', 'h', 't', 'y', 'P']

sorted():

এই মেথডের মাধ্যমে কোন একটা লিস্ট এর আইটেম গুলোকে sort করা যায় সেটা , ছোট থেকে বড় বা বড় থেকে ছোট যেকোন ভাবে হতে পারে |
ছোট থেকে বড় ভাবে সাজানোর জন্য নিচের মত করে কোড লিখা যায় |

lst=[1,2,3,10,5,6]
print(sorted(lst))

Output: [1, 2, 3, 5, 6, 10]


একই ভাবে বড় থেকে ছোট ভাবে সাজানোর জন্য নিচের মত করে কোড লিখা যায় |

lst=[1,2,3,10,5,6]
print(sorted(lst,reverse=True))

Output: [10, 6, 5, 3, 2, 1]
